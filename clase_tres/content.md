## 1. Sets

Los sets (conjuntos) en Python son colecciones desordenadas, mutables y sin elementos duplicados. Se definen usando llaves {} o set(), ideales para eliminar duplicados de listas y realizar operaciones matemáticas de conjuntos como uniones e intersecciones.

## 2. Condicionales

Los condicionales en Python (if, else, elif) son estructuras de control que permiten tomar decisiones y ejecutar bloques de código específicos basados en si una condición es verdadera o falsa. Las simples actúan bajo un solo escenario (if), mientras que las compuestas gestionan múltiples alternativas (if-else, if-elif-else).

## Bucle While

El bucle while en Python es una estructura de control que repite un bloque de código mientras una condición sea verdadera (True). Es ideal cuando no se conoce de antemano cuántas iteraciones se necesitan, deteniéndose automáticamente cuando la condición se vuelve falsa (False).

## Funciones

Las funciones en Python son bloques de código reutilizables y organizados que realizan una tarea específica. Se definen usando la palabra clave def, permitiendo agrupar instrucciones para evitar repeticiones, mejorar la legibilidad y estructurar programas complejos en unidades más sencillas.

### Módulos

Los módulos en Python son archivos con extensión .py que contienen código reutilizable, incluyendo funciones, clases y variables. Permiten organizar programas grandes en partes más pequeñas, facilitando el mantenimiento, la reutilización de código y la gestión de espacios de nombres para evitar conflictos. Se importan utilizando la palabra clave import.
